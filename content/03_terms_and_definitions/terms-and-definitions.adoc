Road::
Roads are the core elements for any road network in ASAM OpenCRG.
Each road runs along one road reference line.
ASAM OpenCRG data is applied to the road surface.

Road elevation::
Road elevation specifies the elevation along the road reference line, that is in s direction.

Road reference line::
A road reference line defines the horizontal alignment of a road and consists of one or more geometry elements.
Every road has one road reference line.
ASAM OpenCRG data is defined relative to the road reference line.

Road reference line coordinate system::
The road reference line is always located within the x/y-plane defined by the inertial coordinate system.
A road reference line coordinate system runs along the road reference line.
It is a right-handed coordinate system.
The u-axis follows the tangent of the road reference line.
The v-axis is orthogonal to the s-axis.
The right-handed coordinate system is completed by defining the up-direction z orthogonal to u-axis and v-axis.
